## 💧 북라커(picogram)


✈ [데모사이트 보러가기](https://book-locker-project.vercel.app/) ✈

<br/>

>
> ###  💡 프로젝트 설명 📝
>

<br/>

  🎁 다양한 도서를 결제하고 소장하고 싶은 마음에서 시작된 프로젝트입니다. 
      

<br/><br/>

>
> ###  ⚙️ 주요 기능 ⚙️
>

<br/>

  ✔️ 회원가입
    
  ✔️ 로그인/로그아웃
    
  ✔️ 장바구니에 담기
  
  ✔️ 도서 검색
  
  ✔️ 간단한 결재 시스템 (충전하기 버튼으로 소지금이 추가됩니다.)
  
  ✔️ 댓글달기
  
  ✔️ 파이어베이즈 데이터 읽기/생성
